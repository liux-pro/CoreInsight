# CoreInsight
CoreInsight 是一款高效的系统监控设备，显示实时 CPU 使用率、温度、内存使用率及其他关键统计数据。它采用 STC32 主控制器和 2.4 英寸的 ST7789 屏幕，通过 USB 接口与其配套软件进行通信。
